## 🛠️ Cómo Desplegar tu Academia
1. Sube todos estos archivos a tu repositorio: `agustino8080/Cursos2026/Automatizacion`.
2. Ve a [Vercel](https://vercel.com) e importa el repositorio.
3. El sistema detectará automáticamente el `index.html` y publicará tu curso como una Web App profesional.

## 🎯 ¿Qué aprenderán los alumnos?
*   **Nivel 1:** Control del sistema operativo, gestión masiva de archivos y lógica de bots.
*   **Nivel 2:** Web Scraping con Selenium, interacción con navegadores y bots de mensajería (Telegram).
*   **Nivel 3:** Integración de Cerebros Artificiales (OpenAI), automatización de interfaces y bots en la nube 24/7.
